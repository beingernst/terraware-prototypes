# Example Dashboard Prototype

## Purpose
Demonstrates the prototype system setup with a simple project dashboard.

## Features
- Page layout with title
- Card component with loading state
- Mock data with useMockData hook
- Stats cards in a responsive grid
- Basic list display with actions

## Components Used
- `Page` - Layout wrapper
- `Card` - Content container with loading state
- `Button` - From @terraware/web-components
- `Typography` - From @mui/material
- `Grid` - From @mui/material

## Usage
Navigate to: http://localhost:5173/prototypes/example-dashboard

## Status
Example only - demonstrates component usage patterns
